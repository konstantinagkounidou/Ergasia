git init: το χρησιμοποιήσαμε για να κάνουμε ένα νέο repository.
Τι κάνει: φτιάχνει το νέο branch ως αντίγραφο του master.

git status: το χρησιμοποιήσαμε για να μάθουμε σε τι κατάσταση βρίσκεται το repository.
Τι κάνει: δείχνει την κατάσταση του κόσμου.

git add: το χρησιμοποιήσαμε για να προσθέσουμε ένα αρχείο.
Τι κάνει: παρακολουθεί ένα νέο αρχείο.

git commit: το χρησιμοποιήσαμε για να δημιουργήσουμε ένα commit.
Τι κάνει: περιλαμβάνει τα περιεχόμενα όλων των αρχείων, καταγράφει μετά-δεδομένα και έχει ένα μοναδικό αναγνωριστικό.

git branch: το χρησιμοποιήσαμε για να δημιουργήσουμε ένα νέο branch.
Τι κάνει: δημιουργεί ένα νέο branch και δείχνει τι branches υπάρχουν.

git clone: το χρησιμοποιήσαμε για να κάνουμε αντίγραφο στον υπολογιστή μας.
Τι κάνει: δημιουργεί αντίγραφο.
